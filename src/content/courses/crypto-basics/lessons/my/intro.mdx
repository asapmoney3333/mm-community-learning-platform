# Crypto ဆိုတာ ဘာလဲ

Crypto ဆိုတာ အင်တာနက်ပေါ်မှာ လည်ပတ်တဲ့ **ဒစ်ဂျစ်တယ်ငွေကြေး** တစ်မျိုးဖြစ်ပြီး စာရွက်ပိုက်ဆံလို မဟုတ်ဘဲ network ပေါ်က record များနဲ့ အလုပ်လုပ်ပါတယ်။

<Callout title="အဓိက အယူအဆ" tone="info">
  Crypto က software, shared record, internet access ကို ပေါင်းစပ်ပြီး အဖွဲ့အစည်းတစ်ခုတည်းကို မမှီခိုဘဲ value ပို့နိုင်အောင် လုပ်ပေးပါတယ်။
</Callout>

## လူတွေ စိတ်ဝင်စားရတဲ့ အကြောင်း

- နယ်စပ်ကျော်ပြီး ပိုမြန်မြန် ပို့နိုင်တယ်။
- ကိုယ့် wallet ကို ကိုယ်တိုင် ထိန်းချုပ်နိုင်တယ်။
- blockchain ပေါ်မှာ transaction record ကို အများနဲ့ မျှဝေစစ်ဆေးနိုင်တယ်။

<StatGrid>
  <StatCard label="ပုံစံ" value="Digital">
    Crypto က coin အစစ်မဟုတ်ဘဲ software နဲ့ network state အဖြစ် ရှိနေပါတယ်။
  </StatCard>
  <StatCard label="စာရင်း" value="Shared">
    Computer အများကြီးက transaction record တစ်ခုတည်းကို စစ်ဆေးနိုင်ပါတယ်။
  </StatCard>
  <StatCard label="အသုံးပြုခွင့်" value="Global">
    Wallet နဲ့ internet ရှိရင် ကမ္ဘာတစ်ဝှမ်းက လူတွေ ပါဝင်နိုင်ပါတယ်။
  </StatCard>
</StatGrid>

## အခြေခံ အသုံးအနှုန်းများ

<Glossary>
  <GlossaryItem term="Blockchain">
    Transaction များကို block အလိုက် စီပြီး သိမ်းထားတဲ့ shared record ဖြစ်ပါတယ်။
  </GlossaryItem>
  <GlossaryItem term="Wallet">
    Crypto address နဲ့ key တွေကို စီမံခန့်ခွဲဖို့ အသုံးပြုတဲ့ app သို့မဟုတ် device ဖြစ်ပါတယ်။
  </GlossaryItem>
  <GlossaryItem term="Private key">
    Fund ကို ထိန်းချုပ်ခွင့်ရှိကြောင်း သက်သေပြတဲ့ လျှို့ဝှက် credential ဖြစ်ပါတယ်။ မျှဝေမပေးသင့်ပါ။
  </GlossaryItem>
</Glossary>

<Callout title="လုံခြုံရေး သတိ" tone="warning">
  Private key သို့မဟုတ် recovery phrase ကို တစ်ယောက်ယောက် ရသွားရင် သင့် fund ကို ထိန်းချုပ်နိုင်တတ်ပါတယ်။
</Callout>

## လွယ်ကူတဲ့ စဉ်းစားပုံ

Crypto ကို financial rail အသစ်တစ်ခုလို တွေးကြည့်နိုင်ပါတယ်။

1. Wallet က transaction တစ်ခု ဖန်တီးပြီး sign လုပ်တယ်။
2. Network က valid ဖြစ်မဖြစ် စစ်ဆေးတယ်။
3. Blockchain က result ကို record လုပ်ပြီး အခြားသူတွေ verify လုပ်နိုင်အောင် ထားတယ်။
